# django-reading-list
A New Reading List App

![Reading List Image](http://i.imgur.com/G3cmd7S.png)


## To Install

```
git clone git@github.com:simonv3/django-reading-list.git
virtualenv .
pip install -r requirements.txt
python manage.py runserver
```
