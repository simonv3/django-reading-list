# django-reading-list
A New Reading List App
![Reading List Image] (http://i.imgur.com/zP2w8t0.png)

## To Install

```
git pull origin git@github.com:simonv3/django-reading-list.git
virtualenv .
pip install -r requirements.txt
python manage.py runserver
```
